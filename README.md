Bienvenidos al repositorio central de **Prompts Docentes** para el programa Dev-Gen. Este repositorio contiene instrucciones (prompts maestros) cuidadosamente diseñadas para configurar modelos de Inteligencia Artificial y convertirlos en **Docentes Universitarios Expertos**.

## 📌 Sobre este repositorio

El objetivo de estos prompts es estructurar pedagógicamente el comportamiento de la IA, asegurando que:
- Aplique un enfoque de **Pareto (80/20)**: priorizando enseñar el 20% de los conceptos que generan el 80% de los resultados en producción.
- Mantenga un nivel académico, técnico y profesional.
- No funcione como un simple asistente resolutivo, sino como un educador que **exige validación del conocimiento** antes de avanzar.
- Fomente el aprendizaje activo, la iteración, el ABP (Aprendizaje Basado en Problemas) y el criterio técnico.

Este repositorio está en **constante evolución**. Iremos incorporando nuevas temáticas y afinando las metodologías a medida que se expanda el programa.

## ⚙️ Instrucciones de Uso

1. Navega hasta la tabla de arriba y elige el prompt que se ajuste a lo que deseas aprender o enseñar.
2. Abre el archivo y copia todo el texto Markdown.
3. Pega el texto en ChatGPT, Claude, Copilot o cualquier plataforma conversacional de IA.
4. Indicarle "Ejecuta el prompt."
5. Seguir las instrucciones del Agente Educativo IA para dar comienzo a la clase.
